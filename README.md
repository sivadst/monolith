# MONOLITH — Brutalist Design Studio

A deploy-ready, single-page Brutalist portfolio template.

## Structure
- `index.html` — Main page
- `css/reset.css` — Brutalist CSS reset
- `css/brutalist.css` — Core styles
- `css/components.css` — Reusable components
- `js/main.js` — Interactions

## Deploy to Vercel
1. Push to GitHub
2. Import to Vercel
3. Set framework preset to "Other"
4. Deploy

## Deploy to Netlify
1. Drag and drop the folder to Netlify Drop
2. Or connect Git repo

## Deploy to GitHub Pages
1. Enable Pages in repo settings
2. Select "Deploy from Branch" → main → / (root)

## Customization
- Replace placeholder text in `.work-card-placeholder` with actual images
- Update colours in CSS custom properties
- Add/remove sections as needed
